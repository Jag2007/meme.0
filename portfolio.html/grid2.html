TEXT BOOK

The front matter is the introductory section of your textbook. If you’re using an authoring platform such as Pressbooks, the system will set up some of these sections for you, including a copyright page and a table of contents. Below is a list of common elements found in a textbook’s front matter, in the order they usually appear: title page verso/copyright page dedication table of contents about the authors list of figures or tables foreword preface acknowledgements introduction. While textbooks should have most of these elements, not every textbook will have all of them, so you only need to include the ones you think are relevant.
